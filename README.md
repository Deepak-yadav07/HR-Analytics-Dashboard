HR Analytics Dashboard 

This project is an HR Analytics Dashboard built in Power BI. It provides a clear and interactive view of workforce data with a special focus on attrition (employees leaving the company). The dashboard helps identify key patterns and insights such as which employees are leaving, from which job roles, education backgrounds, salary ranges, and age groups.

This dashboard provides meaningful insights into employee attrition trends in a very structured and professional way. By analyzing the data, HR managers can:
-Understand which employees are more likely to leave (based on age, salary, education, and job role).
-Gain clarity on the attrition rate and its impact on the organization.
-Use these insights to design better retention strategies and improve overall workforce stability.
